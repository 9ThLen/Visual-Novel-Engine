# Visual Novel Engine Wiki

This wiki is the compact project knowledge base. It keeps only current, useful documentation and cross-links to the canonical docs in the repository.

## Start Here

- `overview.md` gives the short project summary.
- `index.md` lists the active wiki pages and repository docs.
- `SCHEMA.md` describes how wiki pages should be maintained.
- `architecture-reference.md` summarizes the current architecture.

## What Was Removed

The wiki previously contained old session reports, dated fix logs, stale implementation plans, empty placeholder pages, a Python virtual environment, lockfiles, and generated tool artifacts. Those files were removed because they were not current documentation.

## Maintenance Rules

- Keep pages short and current.
- Prefer one canonical page over multiple dated variants.
- Move historical notes into git history instead of keeping them as active docs.
- Do not add dependency folders, generated build output, logs, screenshots, or virtual environments to the wiki.
- Update `index.md` when adding or removing wiki pages.
