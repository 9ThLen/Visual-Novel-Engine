### 1.3. Замінити Math.random() на безпечний генератор (C3)

**Проблема**: `Math.random()` не є криптографічно безпечним.

**Файл**: `lib/id-utils.ts`

**Було**:
```typescript
export function generateId(prefix: string, length = 7): string {
  return `${prefix}_${Date.now()}_${Math.random().toString(36).slice(2, 2 + length)}`;
}
```

**Має стати**:
```typescript
function getSecureRandomString(length: number): string {
  const chars = 'abcdefghijklmnopqrstuvwxyz0123456789';
  const array = new Uint8Array(length);
  crypto.getRandomValues(array);
  return Array.from(array, (byte) => chars[byte % chars.length]).join('');
}

export function generateId(prefix: string, length = 7): string {
  return `${prefix}_${Date.now()}_${getSecureRandomString(length)}`);
}

export function generateAssetId(): string {
  return `asset_${Date.now()}_${getSecureRandomString(6)}`;
}

export function generateStoryId(): string {
  return `story_${Date.now()}_${getSecureRandomString(5)}`;
}
```

**Перевірка**: `npx tsc --noEmit --skipLibCheck` — має пройти успішно.

**Пріоритет**: ВИСОКИЙ
**Час**: 30 хвилин
