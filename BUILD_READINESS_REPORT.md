## Команды для сборки

### Разработка
```bash
# Запуск dev сервера
pnpm dev

# Запуск только Metro
pnpm dev:metro

# Android
pnpm android

# iOS
pnpm ios
```

### Production сборка
```bash
# Установка EAS CLI (если не установлен)
npm install -g eas-cli

# Логин в Expo
eas login

# Конфигурация проекта
eas build:configure

# Сборка для Android
eas build --platform android --profile production

# Сборка для iOS
eas build --platform ios --profile production

# Сборка для обеих платформ
eas build --platform all --profile production
```

### Preview сборка (для тестирования)
```bash
eas build --platform android --profile preview
```
