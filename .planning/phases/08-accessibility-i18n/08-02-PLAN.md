<interfaces>
From hooks/use-colors.ts:
```typescript
export function useColors(colorSchemeOverride?: ColorScheme): ThemeColorPalette
```
Returns a `RuntimePalette` with all tokens. Usage: `const colors = useColors();` then `colors.primary`, `colors.error`, `colors['text-inverse']`, etc.

Available tokens relevant for this plan:
- `colors.primary` ŌĆö main accent (violet ~ #7c5bf5)
- `colors.background` ŌĆö page bg (~ warm beige/dark indigo)
- `colors.surface` ŌĆö card bg
- `colors.foreground` ŌĆö primary text
- `colors.muted` ŌĆö secondary text / foreground-tertiary
- `colors.border` ŌĆö border color
- `colors.error` / `colors.danger` ŌĆö error red (always resolves, no fallback needed)
- `colors.success` ŌĆö success green (always resolves)
- `colors['text-inverse']` ŌĆö white text on primary (fixed by 08-01)
- `colors['surface-container']` ŌĆö elevated surface
- `colors['surface-1']`, `colors['surface-2']` ŌĆö surface elevation levels
</interfaces>
