  <files>__tests__/unit/lib/document-editor.test.ts __tests__/unit/editor/plate-scene-roundtrip.test.ts</files>

    Add or update focused unit tests for:

    - first-use controls marker only on new character
    - stable color across scenes
    - uploaded sprite is attached to the character

    - token keyboard activation opens controls
    - URL/narration colon parsing does not create characters
    - sprite deletion does not physically remove shared assets
    - referenced character deletion shows warning/reference-count behavior
    - sprite names are unique per character but may repeat across characters
    - changing current sprite does not rewrite existing dialogue snapshots
    - oversized image dimensions are rejected or handled safely
    - all new visible strings have i18n keys
