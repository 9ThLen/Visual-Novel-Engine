<task type="auto">
  <name>Implement VN Plate elements</name>
  <files>components/editor/plate/elements/</files>
  <action>
    Add elements for narration/text, dialogue, choice, and technical blocks.

    Element requirements:
    - Every VN element receives or can resolve the canonical `TimelineStep`.
    - Text/dialogue/choice expose editable text fields matching the current
      document-editor behavior.
    - Technical elements include background, character, music, sound,
      transition, variable, effect, camera, and interactive_object.
    - Technical elements render a summary and command label; detailed payload
      editing can remain in the inspector/properties surface.
    - Technical elements persist the full `TimelineStep` payload in the Plate
      node/custom data so serializers never reconstruct defaults.

    Plugin requirements must come from Wave 0. If real Plate requires specific
    plugin registration for custom/void elements, list and implement it here.
  </action>
</task>
