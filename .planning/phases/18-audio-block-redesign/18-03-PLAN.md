<objective>
Remove the "Дія" action dropdown from the music/sound block popover and
replace it with controls for the new contract: a track/silence mode toggle,
separate fade-in/fade-out fields, a scene-vs-continuous binding toggle, and an
optional auto-fade-after field. Update every serialization/validation surface
(Plate bridge, document-editor conversion, text-DSL scene notation) so the new
shape round-trips correctly.
</objective>

reverting the underlying `mode`-based data contract from 18-00.
</rollback>
